A Simple Dart application that uses the terminal prompter to output 
simple questions, you input answers and then it will show your results 
at the end.

Example:
    
    Pick a Color
    [0] - Red
    [1] - Blue
    Enter a Choice:
    
    >1
        
    Do you like Dart (y/n)
    Enter a Choice:

    >y
    
    Results:
    #00f
    true
